# Gesture-Controlled-Interview-Prep
Welcome! This is a hands-free study tool that puts interview questions right onto your webcam feed. You answer them just by holding up your fingers.

The coolest part? It uses old-school, pure math to count your fingers. No heavy AI libraries, no complicated setups, and no installation headaches.
🚀 How to start it up

    Open your terminal in this folder.

    Install the basics: pip install -r requirements.txt

    Run the app: streamlit run app.py

🎮 How to play

    Pick a role from the dropdown to start your camera.

    Put your hand inside the Green Box on the screen.

    Hold up 1, 2, 3, or 4 fingers to pick an answer (A, B, C, or D).

    Hold up 5 fingers (an open hand) to move to the next question.

    To quit, just hit the "Stop" button in the top right corner.
